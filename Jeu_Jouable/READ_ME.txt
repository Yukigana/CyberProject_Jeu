Pour lancer le jeu il faut ouvrir le fichier CyberProject.exe.

Attention ! Il n'est pas possible de fermer le jeu en dehors du menu principal

Seul le personnage de Louis Armstrong est utilisable. 

Touches pour le joueur 1 :
droite : Q
gauche : D
saut : Z
accroupi : S
course : shift gauche
attaque : E


Touches pour le joueur 2 :
droite : croix directionnelle droite
gauche : croix directionnelle gauche
saut : croix directionnelle haut
accroupi : croix directionnelle bas
course : shift droite
attaque : controle droite
